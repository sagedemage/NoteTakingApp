# Frontend

## Build and Run the App with Docker
Build the app
```
docker-compose build
```

Startup the app
```
docker-compose up
```

## Build and Run the App without Docker
### Dependencies
Install dependencies
```
npm install react react-dom react-router-dom react-bootstrap
```

### Run web app
```
npm start
```

### Production

Build the app
```
npm run build
```

Serve the app
```
serve -s build
```
